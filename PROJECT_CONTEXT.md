# MyCamDesk: контекст проекта

Последнее обновление: 2026-06-29.

Рабочая память проекта. Перед изменениями — сверяться. После изменений — обновлять.

## Что это за проект

MyCamDesk — UGC-медиа и комьюнити-платформа для вебкам-индустрии и контент-платформ (OnlyFans, Fansly).

Ориентиры: vc.ru (механика), Т-Ж (подача). Не каталог студий — главная ценность в пользовательском контенте.

## Технический стек

- Next.js 14 App Router + TypeScript
- Prisma + PostgreSQL (локальная на сервере 185.112.83.74)
- NextAuth v5 credentials
- Tailwind CSS + кастомная дизайн-система
- TipTap WYSIWYG
- Resend (email)
- PM2 + Nginx + Let's Encrypt
- GitHub Actions автодеплой (main → prod, dev → test)

## Инфраструктура

- Сервер: 185.112.83.74 (Ubuntu, 2CPU/2GB RAM)
- Production: mycamdesk.com (порт 3000)
- Тестовый: dev.mycamdesk.com (порт 3001)
- DNS: Cloudflare (DNS only, без прокси)
- Файлы: /var/www/uploads/ (Nginx отдаёт, SEO-имена файлов)
- Статика: /var/www/static/_next/static/
- Env: /opt/env.production
- GitHub: znt7cwdygc-crypto/Journal

## Деплой

- Zero-downtime: clone в /opt/site-next → build → swap с /opt/site
- `git push origin main` → production
- `git push origin dev` → тестовый
- .env.local хранится в /opt/env.production (не теряется при swap)

## Реализованные фичи

### Контент
- Статьи с TipTap WYSIWYG, обложками, рубриками, форматами
- Реакции: Нравится, Полезно (компактные pill-кнопки)
- Комментарии с ответами, лайками
- Подписки на авторов и рубрики
- Жалобы на контент, комментарии, профили
- Поделиться — копирование ссылки в буфер

### Вакансии и услуги
- Подробные квизы: вакансия (7 шагов), услуга (8 шагов)
- Публикация на 30 дней, продление из кабинета
- Отзывы на услуги с рейтингом
- Фильтры по городу, категории

### Товары
- До 3 фото на товар (загрузка через /api/uploads)
- Карусель на detail-странице с fullscreen просмотром
- Превью-thumbnail в карточке каталога
- Публикация на 30 дней

### Резюме
- Два типа: модель (12 шагов) и специалист (10 шагов)
- Модель: формат → город+процент → о себе → нагрузка → площадки → деньги → условия → команда → пожелания → контакты → итог
- Специалист: о себе → нагрузка → навыки → деньги → условия → команда → пожелания → контакты → итог
- Публикация на 7 дней

### Платная разблокировка резюме (инвайт-система)
- Модель: $15, Специалист: $5
- Обратный квиз (студия подтверждает требования модели)
- Холдирование → согласие/отказ модели → списание/возврат
- 72 часа на ответ, автоистечение через cron
- Причина отклонения обязательна
- Жалоба на несоответствие условий
- Баланс студии в шапке кабинета

### Модель-оператор
- Связки модель↔оператор
- Квиз с чипами, требованиями, важностью
- Публикация на 14 дней

### Гайды (SEO-лендинги)
- В БД (таблица Guide), управление из админки
- Динамические: создание, редактирование, публикация, отображение на главной
- FAQ JSON-LD

### Полезные ссылки
- В БД (таблица UsefulLink), управление из админки
- 16 ссылок с описаниями, категориями

### Email-уведомления (Resend)
- Подтверждение email при регистрации
- Инвайт получен → модели
- Инвайт принят → обоим (контакты)
- Инвайт отклонён → студии (причина)
- Баланс пополнен → студии

### Админка (отдельный layout, 11 страниц)
- Дашборд: статистика
- Пользователи: поиск, роли (USER/MODERATOR/ADMIN), блокировка
- Контент: вкладки, инлайн-редактирование, статусы
- Жалобы: фильтры, скрыть/разрешить
- Инвайты: все статусы, доход
- Реклама: создание, редактирование, toggle
- Отзывы: удаление
- Платежи: approve/reject
- Баланс: пополнение в долларах, все балансы, транзакции
- Аудит-лог: все действия админов
- Гайды: CRUD
- Ссылки: CRUD

### SEO
- JSON-LD: WebSite, Organization, CollectionPage, Article, BreadcrumbList, FAQPage, Product, ProfilePage
- OG-image автогенерация
- Favicon SVG
- Sitemap.xml динамический (force-dynamic, revalidate 1h)
- robots.txt с AI-ботами
- llms.txt
- IndexNow API
- Мета-теги на всех страницах

### Поиск
- По статьям, авторам, вакансиям, услугам, товарам, резюме, гайдам, полезным ссылкам
- Поисковые запросы сохраняются в SearchQuery
- Популярные запросы в боковой панели (реальные данные)

### Реклама
- 11 placement'ов (главная, лента, каталоги, сайдбар)
- Управление из админки
- Сайдбар-реклама на мобильном показывается на главной

### Загрузка файлов
- Сохранение на диск (/var/www/uploads/)
- SEO-имена: транслит заголовка + UUID
- Nginx отдаёт напрямую с кэшем 30 дней
- /media/ → alias на /uploads/ (обратная совместимость)
- Аватары, обложки статей, фото товаров

### Тесты
- QA-скрипт: scripts/role-scenarios-qa.mjs (56 проверок, 9 типов пользователей)
- Playwright E2E: 163 теста на 3 устройствах (Desktop Chrome, iPhone 14, Safari WebKit)

## Принятые решения (не менять)

- Репост в ленте убран. Репост только на странице статьи — копирование ссылки
- Автор в карточке ленты только сверху
- Рубрика статьи обязательна
- Контакты резюме платные (инвайт-система)
- Демо-статьи удалены — пустая лента показывает заглушку
- Формат студии — первый вопрос в квизе модели
- Город показывается только для офлайн/любой формат
- Баланс интегрирован в шапку кабинета (нет отдельного блока)
- Профиль редактируется в шапке через ⚙ (нет отдельного блока)

## Как работать

1. Клонировать: `git clone https://github.com/znt7cwdygc-crypto/Journal.git`
2. Установить: `npm install && npx prisma generate`
3. Env: скопировать .env.local с сервера или создать с DATABASE_URL, AUTH_SECRET, NEXTAUTH_URL, NEXT_PUBLIC_SITE_URL, RESEND_API_KEY
4. Dev: `npm run dev` (или `npm run build && npm start`)
5. Ветка `dev` для тестирования, `main` для production
6. После изменений: `npx tsc --noEmit` для проверки
